# Product Card ‚Äì Interactive UI

This repository contains an **interactive product card** designed to display essential product information in a clear and engaging way.

The product card includes:
- brand name
- product model
- price
- size selection
- an "Add to cart" button

When the user clicks on the call-to-action button, a **validation popup** is displayed to confirm the action, enhancing the user experience and interaction feedback.

The interface is built using **HTML, CSS, and JavaScript**, with a strong focus on visual presentation and smooth interactions.
Animations are handled using **CSS**, while **JavaScript** is used to manage user interactions and calls-to-action.
